# AlgVex

AI 辅助的量化研究与自动化系统，基于 GitHub + Claude Code + Slack 集成。

## 功能模块

- **claude.yml** - @claude 交互响应
- **pr-review.yml** - PR 自动代码审查
- **issue-triage.yml** - Issue 自动分类
- **issue-deduplication.yml** - Issue 重复检测
- **ci-failure-auto-fix.yml** - CI 失败自动修复
- **test-failure-analysis.yml** - 测试失败分析
- **manual-code-analysis.yml** - 手动代码分析

## 使用方式

在 Issue 或 PR 中 @claude 即可触发 AI 助手。
